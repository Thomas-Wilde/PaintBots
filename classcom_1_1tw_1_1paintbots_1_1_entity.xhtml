Detailed Description

Basic class of entities/objects, that belong to the game. Basically everything that the GameManager manages is an Entity.

Set the position of the Entity. The position has to be on the gaming board, i.e. in the interval [x,y] = [0, width) x [0,height), were width and height are the board dimensions in game coordinates. Only the GameManager can call this method.

Returns
false if the position is outside of the board, true otherwise.
