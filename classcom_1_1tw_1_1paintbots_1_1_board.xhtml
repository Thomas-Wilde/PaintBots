Detailed Description

The Board represents the area on which the players move around, i.e. the game board. It contains different areas and gives information, e.g. about obstacles, paintable areas, and possible interactions. The Board has the same dimensions as the Canvas. It is split into different cells, e.g. the pixels that can be painted.
