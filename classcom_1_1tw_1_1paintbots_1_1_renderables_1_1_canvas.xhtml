Detailed Description

The Canvas represents the area that gets painted by the players. It has the same size as the board. The paint process is performed in circles with a given radius. The Canvas holds a Pixmap that is used for the painting. The Pixmap gets is send to a texture that is drawn to the screen. The canvas also counts how many pixels/cells are covered with each color.

Paint the canvas at the given position, with the given color, if it is possible to paint there. The paint is applied in circles. This method is only available to the GameManager.

Parameters
